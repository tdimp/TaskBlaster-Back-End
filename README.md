# Phase 3 Project Build 3 Backend
This is the backend for a task lister app created for
Flatiron School to showcase skills in Ruby and ActiveRecord.

### Requirements

Ruby 2.7.4
SQLite 3.31.1

### How to use
Fork and clone
run bundle install
run rake db:migrate
run rake server
follow instructions on frontend
